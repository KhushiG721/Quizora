# Quizora

## Overview

Quizora is a Multi-User Online Quiz and Test Management System developed in C++. It provides separate interfaces for administrators and students, enabling quiz creation, management, participation, and result evaluation.

## Features

### Admin Module
- Admin Login
- Create Quizzes
- Add Questions
- Manage Users
- View Results

### Student Module
- Student Login
- Attempt Quizzes
- View Scores
- Performance Evaluation

- Data Structures
